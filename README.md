# Zubair Chat App

A simple standalone chat interface built with HTML, CSS, and JavaScript.

## Files

- `index.html` — the main chat page
- `css/styles.css` — styling for the chat UI
- `js/script.js` — chat behavior and send button handling

## Usage

Open `index.html` directly in your browser:

- Double-click `index.html`
- Or open it with your browser using `File > Open`

The page supports two chat panels side by side. Type a message and press Enter or click Send.

## Notes

- This version works without a server.
- The chat is local to your browser window and does not persist messages after refresh.
